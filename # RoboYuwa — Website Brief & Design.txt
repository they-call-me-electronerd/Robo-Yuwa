# RoboYuwa — Website Brief & Design Specification (Updated for HTML, CSS, JavaScript Only)

**Purpose:** Build a modern, professional, SEO-friendly website for RoboYuwa — a nonprofit NGO that guides the next generation in Robotics, AI, and IoT. The site should be futuristic, clean, and optimized using only HTML, CSS, and JavaScript.

---

## 1. Project Overview

* **Project name:** RoboYuwa
* **Organization type:** Non-profit / NGO
* **Primary goals:**
  * Showcase mission, programs, projects, and events
  * Recruit students, volunteers, and mentors
  * Attract sponsors and donations
  * Publish tutorials, blogs, and project case studies
  * Provide easy contact and registration flows

---

## 2. Target Audiences

1. Students (12–24) curious about Robotics, AI, IoT  
2. Schools & educators  
3. Sponsors and donors  
4. Volunteers and mentors  
5. Media and general public  

---

## 3. Sitemap

* Home  
* About (Mission, Vision, Team, Partners)  
* Programs (Workshops, Courses, Mentorship)  
* Projects (Gallery, Categories, Case Studies)  
* Events (Upcoming + Registration)  
* Blog / Tutorials  
* Get Involved (Volunteer, Mentor, Sponsor)  
* Donate  
* Contact  
* Privacy Policy  

---

## 4. Features

* Responsive design using CSS Grid + Flexbox  
* High performance (lazy loading, WebP, optimized JS)  
* SEO-ready (meta tags, structured data, sitemaps)  
* Accessible (WCAG AA basics)  
* Event registration using forms + JS (fetch API)  
* Dynamic project gallery (JavaScript filtering + lightbox)  
* Newsletter signup  
* Analytics: GA4 + Search Console  

---

## 5. UX / UI Guidelines

**Tone:** Clean, futuristic, professional.

**Visual direction:**  
* Tech-themed hero with subtle animations  
* Soft glows, neon lines, card-based layout  
* Hover micro-interactions  
* Honeycomb pattern optional background  
* ESP32 illustration or 3D render on homepage hero  

**Navigation:**  
* Sticky navbar  
* Mobile hamburger menu written in JS  

---

## 6. Design System

**Colors (example):**  
* Primary: #0F172A (navy)  
* Accent: #00A3FF (blue neon)  
* Secondary: #7C3AED (violet)  
* Neutral: #F8FAFC, #64748B  

**Typography:**  
* Headings: Poppins / Inter  
* Body: Roboto / Open Sans  

**Layout:**  
* 8px spacing grid  
* Card radius: 12px  
* Full-width hero section  

---

## 7. Frontend Tech Stack (Updated — No Frameworks)

**Core:**  
* HTML5  
* CSS3 (Flexbox, Grid, animations)  
* JavaScript (Vanilla JS only)

**Optional lightweight tools:**  
* CSS Variables  
* CSS keyframe animations  
* Fetch API for forms and dynamic loading  
* SVG/WebP assets  
* Vite (optional) for fast development  

**Folder Structure:**

```
/assets
  /images
  /icons
/css
  styles.css
/js
  main.js
  animations.js
  forms.js
/pages
  about.html
  programs.html
  projects.html
  events.html
  blog.html
index.html
```

---

## 8. SEO Strategy

* Use semantic HTML structure  
* Unique meta title + description for each page  
* JSON-LD schema for Organization, Events, Blog  
* Lazy-load images  
* ALT text everywhere  
* Robots.txt + sitemap.xml  

---

## 9. Accessibility

* High contrast  
* Keyboard navigable menus  
* ARIA labels  
* Descriptive links  
* Form validation with JavaScript
